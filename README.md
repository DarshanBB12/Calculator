# Calculator

# Calculator Project (HTML, CSS & JavaScript)

This project is a Simple Calculator created using HTML, CSS, and JavaScript.
It is a front-end application designed to perform basic arithmetic operations while practicing DOM manipulation and event handling.

# Project Description

The goal of this project is to build a functional calculator with a clean and modern user interface.
It focuses on handling user input, performing calculations dynamically, and updating the display using JavaScript.

The calculator supports number inputs, operators, and special functions like clear and delete.


# Features

~ Basic arithmetic operations (+, −, ×, ÷)
~ Percentage (%) support
~ Clear All (AC) button
~ Delete last character (SC)
~ Equal (=) to evaluate expression
~ Error handling for invalid inputs
~ Clean and modern dark UI design

# Technologies Used

~HTML5
~CSS3
~JavaScript (DOM Manipulation & Events)
